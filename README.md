# Универсальный конвертер изображений
Позволяет выполнять различные операции с изображениями, такие как горизонтальное и вертикальное отражение, выделение границ с помощью оператора Собеля, а также конвертацию изображений между поддерживаемыми форматами.

Основные возможности:  
- Горизонтальное отражение: инвертирует изображение по горизонтальной оси.
- Вертикальное отражение: инвертирует изображение по вертикальной оси.
- Оператор Собеля: выделяет границы и резкие переходы на изображении.
- Конвертирование изображений: поддерживает конвертацию изображений между форматами PPM, BMP, PNG.
